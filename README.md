# msgroup - Landing Page

Repositorio oficial de la landing page corporativa de msgroup. Este proyecto funciona exclusivamente como una página de aterrizaje estática e informativa para la presentación de servicios de producción técnica, iluminación, audio y pantallas LED, optimizada para la captura de clientes potenciales.

## Arquitectura y Stack Técnico

* **Framework:** Angular (Versión 17+) con Server-Side Rendering (SSR) básico para indexación SEO.
* **Estilos:** Tailwind CSS / SCSS.
* **Interactividad:** Enrutamiento directo mediante enlaces parametrizados hacia la API de WhatsApp, delegando la lógica de negocio y conversión fuera de la aplicación.

## Características del Proyecto

* Interfaz de usuario de una sola página (Single Page) sin estado complejo ni almacenamiento de datos local.
* Componentes presentacionales diseñados para el despliegue rápido de información comercial y galerías multimedia.
* Optimización de rendimiento orientada a Core Web Vitals para dispositivos móviles.
